AniMotto API docs

The Base URL (Base_URL) for all API endpoints is: http://localhost:8000/api

Here you will find all the currently available resources and API endpoints from AniMotto. The API project is still under active development so expect to see new changes and features every now and then.

Above you will find the base url for this API. There will be more than one endpoint that will be available for use. Click through the sidebar navigation to see the different endpoints and their explanations of how to use them. If you want to know where you are currently, you are on the overview.

The Quote object

The is the JSON structure that gets returned for all /quotes endpoints. 


{
    quote: "The strong should aid and protect the weak. Then, the weak will become strong, and they in turn will aid and protect those weaker than them. That is the law of nature.",
    anime: {
        name: "demon slayer",
        altName: "kimetsu no yaiba",
    },
    character: "tanjiro kamado",
}

Get a random quote

This endpoint responds with a random quote from any anime/character.

The endpoint:

/quotes/random

Return

It will return a Quote object

Example:

Here, we are using the global web API called fetch, which is both available in browser and NodeJS as of now. But you can use any programming language of your choosing. 


const response = await fetch(`${Base_URL}/quotes/random`)
const data = await response.json()

Get a random quote via Anime name

This endpoint returns a random quote from a given Anime name.

The endpoint:

/quotes/random?anime=(anime_name)

Return

It will return a Quote object

Example:

Here, we are using the global web API called fetch, which is both available in browser and NodeJS as of now. But you can use any programming language of your choosing. 


const response = await fetch(`${Base_URL}/quotes/random?anime=one piece`)
const data = await response.json()
