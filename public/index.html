AniMotto

An API for Quotes from Various Anime from your Favorite Anime Characters

Explore a rich collection of anime quotes.
Perfect for developers, fans, and enthusiasts.

(Don't worry, it's not blocked by CORS)

View API Info View Documentation

Example Usage:

The below piece of code returns a random quote.


fetch('http://localhost:8000/api/quotes/random')
    .then(response =>response.json())
    .then(quote => console.log(quote))
